# 交易计划示例

本文档提供了几个完整的交易计划示例，帮助用户理解如何创建自己的交易计划。

## 示例1：短线趋势追踪系统

### 基本信息
- **系统名称：** 短线趋势追踪系统
- **交易风格：** 短线（1-5天）
- **风险偏好：** 激进
- **系统描述：** 基于技术面分析，捕捉短期趋势机会

### 选股规则

**必须满足条件：**
1. 市值 > 50亿
2. 流通盘 > 10亿
3. 近3日平均成交额 > 1亿
4. 股价在20日均线之上
5. 近5日涨幅 > 5%

**排除条件：**
1. ST股票
2. 次新股（上市不足1年）
3. 连续跌停
4. 近期有重大利空消息

### 择时规则

**入场信号：**
1. 大盘处于上升趋势（沪深300指数MA20 > MA60）
2. 股价突破前期高点
3. MACD金叉
4. 成交量较前5日平均放大50%以上
5. 连续3日收阳

### 仓位规则
- **单只股票仓位上限：** 15%
- **最大持股数：** 8只
- **最小持股数：** 3只

### 风险管理
- **止损类型：** 固定比例止损
- **止损比例：** 7%
- **止盈类型：** 移动止盈
- **止盈条件：** 跌破5日均线

### 纪律规则

**必须做到：**
1. 严格执行止损，不得有任何借口
2. 每次交易前必须检查所有选股条件
3. 盈利后及时复盘总结
4. 每日收盘后更新交易日志

**绝对禁止：**
1. 追涨杀跌，情绪化交易
2. 重仓单只股票超过15%
3. 一天内频繁交易同一只股票
4. 不设止损就入场

## 示例2：中线价值成长系统

### 基本信息
- **系统名称：** 中线价值成长系统
- **交易风格：** 中线（1-3个月）
- **风险偏好：** 中性
- **系统描述：** 结合基本面和技术面，寻找价值被低估的成长股

### 选股规则

**必须满足条件：**
1. 市值 > 100亿
2. ROE > 15%（最近3年平均）
3. 营收增长率 > 20%（最近3年平均）
4. 市盈率 < 行业平均
5. 负债率 < 60%

**排除条件：**
1. ST股票
2. 业绩连续下滑
3. 有重大诉讼或处罚
4. 大股东减持

### 择时规则

**入场信号：**
1. 大盘处于震荡或上升趋势
2. 股价回调至60日均线附近
3. MACD即将金叉或刚刚金叉
4. 出现重大利好消息（如业绩超预期）
5. 机构资金流入

### 仓位规则
- **单只股票仓位上限：** 20%
- **最大持股数：** 6只
- **最小持股数：** 3只

### 风险管理
- **止损类型：** 固定比例止损
- **止损比例：** 12%
- **止盈类型：** 分批止盈
- **止盈策略：** 涨30%卖1/3，涨50%卖1/3，剩余持有

### 纪律规则

**必须做到：**
1. 每季度财报发布后重新评估持仓
2. 严格执行止损和止盈计划
3. 定期复盘，每月至少一次
4. 保持仓位分散，不超过3个行业

**绝对禁止：**
1. 因短期波动而频繁交易
2. 不做基本面研究就买入
3. 重仓单只股票
4. 追高买入

## 示例3：长线价值投资系统

### 基本信息
- **系统名称：** 长线价值投资系统
- **交易风格：** 长线（3个月以上）
- **风险偏好：** 保守
- **系统描述：** 专注于优质蓝筹股，长期持有，分享企业成长

### 选股规则

**必须满足条件：**
1. 市值 > 500亿
2. ROE > 15%（最近5年平均）
3. 连续5年盈利增长
4. 行业龙头地位
5. 分红率 > 2%

**排除条件：**
1. 周期性行业（除非在底部）
2. 高负债企业（负债率 > 70%）
3. 管理层频繁变动
4. 主营业务不清晰

### 择时规则

**入场信号：**
1. 股价处于历史估值低位（PE < 历史平均）
2. 出现系统性风险导致的非理性下跌
3. 重大利好（如政策支持、行业拐点）
4. 大股东或管理层增持

### 仓位规则
- **单只股票仓位上限：** 30%
- **最大持股数：** 5只
- **最小持股数：** 2只

### 风险管理
- **止损类型：** 技术位止损
- **止损条件：** 跌破年线且基本面恶化
- **止盈类型：** 不设固定止盈
- **止盈条件：** 估值过高或基本面恶化时卖出

### 纪律规则

**必须做到：**
1. 每年至少深度研究一次持仓公司
2. 关注公司重大公告和财报
3. 保持耐心，不因短期波动而动摇
4. 定期再平衡，维持合理仓位

**绝对禁止：**
1. 因短期涨跌而频繁交易
2. 追逐热点和概念股
3. 使用杠杆
4. 不做研究就买入
