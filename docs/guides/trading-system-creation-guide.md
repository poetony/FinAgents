# 交易计划创建向导说明

## 概述

为了帮助用户更好地创建交易计划，我们在创建向导的每个步骤都添加了详细的引导说明和示例。

## 各步骤引导内容

### 第1步：基本信息

**引导说明：**
- 交易计划是交易决策的核心框架，帮助保持纪律性和一致性

**交易风格说明：**
- **短线：** 持仓1-5天，关注技术面和市场情绪，适合有时间盯盘的投资者
- **中线：** 持仓1-3个月，结合技术面和基本面，适合上班族
- **长线：** 持仓3个月以上，重点关注基本面，适合价值投资者

### 第2步：选股规则

**引导说明：**
- 定义选股标准，系统会在复盘时检查是否符合这些条件

**示例：**
- **必须满足：** 市值 > 50亿、流通盘 > 10亿、近3日成交额 > 1亿
- **排除条件：** ST股票、次新股（上市不足1年）、连续跌停

**输入框提示：**
- 必须满足条件：例如：市值 > 50亿
- 排除条件：例如：ST股票、次新股

### 第3步：择时规则

**引导说明：**
- 定义入场时机，明确什么情况下可以买入

**示例：**
- **市场环境：** 大盘处于上升趋势（MA20 > MA60）
- **技术信号：** 股价突破前期高点、MACD金叉、成交量放大
- **确认条件：** 连续3日收阳、突破后回踩不破支撑

**输入框提示：**
- 入场信号：例如：股价突破20日均线且成交量放大

### 第4步：仓位规则

**引导说明：**
- 合理的仓位管理是风险控制的关键

**建议：**
- **单只股票仓位：** 短线 10-15%，中线 15-20%，长线 20-30%
- **持股数量：** 短线 5-10只，中线 3-8只，长线 2-5只
- **分散原则：** 不同行业、不同板块，避免集中风险

### 第5步：风险管理

**引导说明：**
- 设置止损和止盈规则，保护资金安全

**建议：**
- **止损：** 短线 5-8%，中线 10-15%，长线 20-25%
- **止盈：** 可以使用移动止盈（如跌破5日线）或分批止盈（涨20%卖1/3，涨40%卖1/3）
- **盈亏比：** 建议至少 1:2（止损10%，止盈至少20%）

### 第6步：纪律规则

**引导说明：**
- 明确交易纪律，这是交易计划的灵魂

**示例：**
- **必须做到：** 严格执行止损、每次交易前检查选股条件、盈利后及时复盘总结
- **严禁操作：** 追涨杀跌、重仓单只股票、频繁交易、情绪化交易
- **违规处理：** 暂停交易1周、减少仓位、强制复盘

**输入框提示：**
- 必须做到：例如：严格执行止损、每次交易前检查选股条件
- 绝对禁止：例如：追涨杀跌、重仓单只股票、频繁交易

### 第7步：完成确认

**引导说明：**
- 恭喜即将完成交易计划的创建

**下一步建议：**
- **开始复盘：** 使用交易计划对历史行情进行复盘，验证系统的有效性
- **持续优化：** 根据复盘结果不断调整和完善交易规则
- **严格执行：** 在实际交易中严格遵守系统规则，保持纪律性

## 设计原则

1. **渐进式引导：** 每个步骤都有清晰的说明和示例
2. **实用性：** 提供具体的数值建议和操作指南
3. **可视化：** 使用不同颜色的提示框区分不同类型的信息
4. **灵活性：** 用户可以根据自己的情况调整参数

## 用户体验优化

1. **颜色编码：**
   - 基本信息：蓝色（info）
   - 选股规则：蓝色（info）
   - 择时规则：蓝色（info）
   - 仓位规则：绿色（success）
   - 风险管理：橙色（warning）
   - 纪律规则：红色（error）
   - 完成确认：绿色（success）

2. **示例丰富：** 每个步骤都提供了具体的示例，帮助用户理解
3. **提示明确：** 输入框的placeholder提供了具体的示例格式
4. **建议合理：** 根据不同的交易风格提供不同的参数建议

## 技术实现

- 使用 `el-alert` 组件展示引导说明
- 使用自定义样式美化示例区域
- 保持与整体UI风格的一致性
- 响应式设计，适配不同屏幕尺寸
