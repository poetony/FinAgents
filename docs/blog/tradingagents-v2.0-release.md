# TradingAgentsCN v2.0 预告：老用户值得期待的三大升级

> 这不是一篇“发布公告”，而是一封写给老用户的“剧透信”

---

## 🎉 写在前面

过去这一年，不少朋友已经在用 TradingAgentsCN 做**单股分析、批量分析**，让 AI 帮忙看盘、读公告、拆研报。

v2.0 我们没有去堆“更多功能”，而是围绕一个核心问题打磨：

> **“怎么让 AI 分析，真正融入你的交易决策和复盘流程？”**

这篇文章，想提前跟老用户剧透一下 v2.0 的三大新能力：

- 🧠 更聪明、可定制的 **统一分析引擎 + 工作流**
- 📦 直接面向真实账户的 **持仓分析**
- 📒 真正落在交易行为上的 **操作复盘与阶段性复盘**

预计会在 **元旦假期后** 开始灰度开放 v2.0，下面是你可以期待的变化。

---

## 🔧 1. 统一分析引擎 + 可视化工作流（升级你的“分析大脑”）

在 1.x 版本里，分析流程更多是「写死」在代码里的：

- 每一步的顺序、参与的智能体、提示词和参数都是固定的；
- 想改流程，只能改代码、重新部署；
- 用户很难按自己的交易习惯做细节上的裁剪。

结果就是：**想做一点个性化调整都很费劲，前端入口也只能适配少数几种固定玩法。**

v2.0 做了一件“看起来不起眼，但对整个系统非常关键”的事：

> **把原来写死的分析链路，抽象成一套由可视化工作流驱动的统一分析引擎，
> 用户可以根据自己的需要，自由增减节点、调整顺序，对整个分析流程做裁剪。**

### 1.1 对你有什么直接好处？

1. **分析结果更稳定、一致**  
   不管是从“单股分析”、还是“批量分析”、还是“工作流执行”进去，
   背后跑的都是同一套工作流，
   - 目标价格、参考价的生成逻辑一致
   - 多头/空头/中性观点的结构一致
   - 风险评估、信心指数等字段不会“这有那边没有”

2. **可以“看得见”的分析过程**  
   每次分析，不再是一个黑盒：
   - 每个节点的输出都可以单独查看
   - 哪一步没跑通，一眼就能看出来
   - 出现问题时，能精确到“是哪个智能体没想明白”

3. **旧引擎不会马上消失**  
   - 会保留“旧引擎 vs 新引擎”的切换选项
   - 新版默认推荐使用统一分析引擎
   - 遇到极端情况，你仍然可以一键切回老流程

### 1.2 可视化工作流：把“你的分析方法”变成一张图

v2.0 的工作流编辑器，也做了一轮比较大的升级：

- 支持把“从分析师到研究员辩论，再到交易员决策、风控”的全过程，
  **完整拉成一张图**；
- 每个节点都可以单独配置提示词、模型、参数；
- 支持多头/空头/中性研究员多轮辩论；
- 支持风险节点单独插入，而不是“写死”在最后。

### 1.3 大盘 / 行业分析：给个股一个“位置感”

v2.0 在默认分析流程里，新加了**大盘分析节点**和**行业分析节点**，
不再只盯着单只股票本身，而是先回答两个问题：

- 现在整个市场（大盘）是偏风险开还是风险关？
- 这只股票所在行业，目前处在景气周期的什么阶段？

这些信息会反馈到后面的研究员辩论和交易决策节点里，
典型的效果包括：

- 在大盘高位、整体风险偏高的时候，对激进的“满仓梭哈型”建议自动降权；
- 行业景气度下行时，对“基本面看起来还可以”的个股，给出更保守的目标价和仓位建议；
- 同一只股票，在不同行情阶段下，给出**更有“位置感”的建议**（而不是只看静态指标）。

简单来说：

> **大盘分析 + 行业分析，让每一份个股结论，都有了清晰的市场坐标系，
> 提升了整个分析链路的“方向感”和准确性。**

## 📦 2. 持仓分析：从“看股票”走向“看组合”

1.x 版本更多还是在帮你“看一只股票好不好”。

到了 v2.0，我们希望往前走半步：

> **不只是分析一只股票，而是帮你看看——整个仓位到底合不合理？**

### 2.1 组合层面的健康体检

v2.0 新增了“持仓分析”模块（专业版功能），
可以针对你的真实/模拟账户，做一份**组合体检报告**：

- 当前总市值、总成本、浮动盈亏
- 行业分布、个股集中度
- 前 1 / 前 3 / 前 5 大持仓占比
- 简单易懂的风险等级（高 / 中 / 低）

**[图片位置：组合持仓分析总览截图]**

你可以把它理解为：

> “帮你把账户里的每一笔仓位，投到一张风险雷达图里，
> 看看有没有哪一块『过于突出』。”

### 2.2 单只持仓的个性化诊断

和普通“单股分析”不同，单只持仓分析会额外考虑：

- 你的买入成本
- 你的持仓天数
- 这只股票在你总资产中的权重

然后基于这些信息，给出更贴近你实际情况的判断：

- 现在这个价位，基于你的成本，**是加仓、减仓，还是应该果断止损？**
- 当前位置对应的盈亏空间，**和历史波动相比处于什么水平？**
- 这只股票在你账户里的仓位，**是不是已经高到让你睡不安稳？**

**[图片位置：单只持仓分析报告截图]**

这类分析的一个核心思路是：

> **不是帮你判断“这只股票好不好”，
> 而是帮你判断“你这笔持仓，还值不值得继续拿着”。**

## 📒 3. 交易复盘：把经验沉淀成“自己的案例库”

另一个很多老用户提过的需求是：

> “AI 能不能帮我复盘自己的交易，而不是只看标的？”

v2.0 里，我们围绕这个场景做了一整套“操作复盘”能力（专业版）：

### 3.1 单笔 / 完整交易复盘

从模拟交易记录里，选中一笔或一轮完整交易，系统会自动：

- 把买入到卖出的完整 K 线走势拉出来；
- 计算最优买卖点、最大回撤、错失收益；
- 用大模型从“纪律执行、情绪波动、仓位控制”等角度给出点评。

**[图片位置：单笔交易复盘详情页截图]**

适合用来回答那几个经典问题：

- 这笔亏损，**到底是运气问题，还是决策问题？**
- 当时**要是严格按计划执行**，结果会好多少？

### 3.2 周/月/季度的阶段性复盘

除了单笔交易，我们更建议大家养成一个习惯：

> 每个自然周期（周 / 月 / 季度），
> 用 10 分钟做一次“阶段性复盘”。

系统会自动：

- 汇总这段时间所有交易记录；
- 统计：胜率、盈亏比、平均盈利/亏损、最大回撤、总手续费等；
- 让大模型从“交易风格、常见错误、可改进点、下阶段行动计划”等维度给出一份结构化报告。

**[图片位置：阶段性复盘报告截图]**

这部分的目标很简单：

> 帮你在一堆分散的盈亏数字背后，
> 找到那些反复出现的“好习惯”和“坏习惯”。

### 3.3 个人案例库

复盘报告可以一键保存为“案例”，
你可以：

- 给案例打标签（比如：追高、止损犹豫、过早止盈…）
- 添加自己的复盘笔记
- 随时回顾，形成一套“只属于自己的交易教科书”

## 🧩 4. 这些“你可能注意不到，但会悄悄受益”的改动

除了上面这三大块，还有一些偏“工程向”的优化，
对老用户来说感知可能是：

- “怎么感觉报告更完整了？”
- “以前偶尔出错的地方，现在稳定多了。”

主要包括：

1. **统一分析引擎下的字段对齐**  
   - 目标价格、信心指数、风险评分等字段，
     在不同入口下的含义和来源统一了；
   - 多头 / 空头 / 中性观点的历史记录，
     通过更智能的状态合并，避免了“被后面的空值覆盖掉”的问题。

2. **配置获取统一**  
   - 单股分析、批量分析、工作流执行，
     默认都会从同一套“用户偏好 + 系统配置”里读：
     - 默认分析深度
     - 默认分析师组合
     - 默认快速/深度分析模型

3. **报告格式清理**  
   - 对大模型输出做了 Markdown 清理和结构化提取；
   - 让你在前端看到的分析报告，更“像一篇报告”，而不是一大段无格式的聊天记录。

这些东西可能不会出现在宣传页上，
但都直接对应了我们在和大家联调时遇到的那些“小坑”。
