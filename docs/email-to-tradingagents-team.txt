二、情况说明 / Sustainability Challenge & Pro Version
--------------------------------------------------------------------------------

坦诚地说，我们面临项目可持续发展的挑战。

运营这个项目需要持续的成本支出：
- 服务器和 API 费用
- 持续的开发和维护
- 用户支持和文档编写

没有任何收入来源的情况下，项目很难持续发展，也很难为用户提供高质量的服务。

经过慎重考虑，我们决定推出 TradingAgents-CN Pro：

| 版本            | 功能                    | 定价        |
|----------------|------------------------|------------|
| 免费版（开源）   | 核心分析功能、基础工作流   | 永久免费    |
| Pro版（高级）   | 高级功能、优先支持        | 打赏解锁    |

我们想强调：
✅ 开源版本将继续维护
✅ 核心功能对所有用户保持免费
✅ Pro 功能是可选的增强功能

这不是为了盈利，而是为了让项目能够可持续发展，继续为社区服务。

To be transparent, we are facing a sustainability challenge. Running the project requires 
ongoing costs (server, API, development, support). Without any revenue stream, it has 
become difficult to sustain the project's growth.

We have decided to launch TradingAgents-CN Pro:
- Free (Open Source): Core features, free forever
- Pro (Premium): Advanced features, donation-based unlock

We want to emphasize:
✅ The open-source version will continue to be maintained
✅ Core functionality remains free for all users
✅ Pro features are optional enhancements

This is not about profit-seeking, but about making the project sustainable.
