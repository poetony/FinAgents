### 20.1 中长线投资的特点

#### 20.1.1 什么是中长线投资？

**中长线投资定义**：持股周期 1个月 - 1年+

```
中长线投资的时间框架
│
├── 中线投资（1-3个月）
│   ├── 持股1-3个月
│   ├── 捕捉中期趋势
│   └── 兼顾技术面和基本面
│
├── 长线投资（3-12个月）
│   ├── 持股3-12个月
│   ├── 关注基本面和价值
│   └── 等待价值回归
│
└── 超长线投资（1年以上）
    ├── 持股1年以上
    ├── 纯价值投资
    └── 与公司共同成长
```

#### 20.1.2 中长线投资的特点

```
中长线投资的特点
│
├── 价值为本
│   ├── 依赖基本面分析
│   ├── 关注公司价值
│   └── 技术面权重较低
│
├── 时间灵活
│   ├── 不需要频繁盯盘
│   ├── 每周或每月检视即可
│   └── 适合上班族
│
├── 心理轻松
│   ├── 不受短期波动影响
│   ├── 心理压力相对较小
│   └── 更容易坚持
│
├── 需要耐心
│   ├── 价值回归需要时间
│   ├── 可能经历较长的等待期
│   └── 不能急于求成
│
└── 收益稳定
    ├── 长期来看收益更稳定
    ├── 复利效应明显
    └── 适合长期财富积累
```

#### 20.1.3 中长线投资适合什么人？

| 适合 | 不适合 |
|------|--------|
| 没时间盯盘 | 喜欢频繁交易 |
| 有耐心 | 急于求成 |
| 风险承受能力中等 | 追求短期暴利 |
| 追求稳定收益 | 无法承受中期波动 |
| 有一定基本面分析能力 | 完全不懂基本面 |

### 20.4 中长线投资的注意事项

#### 20.4.1 中长线投资的五大原则

```
┌─────────────────────────────────────────────────────────────┐
│                   中长线投资五大原则                          │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   原则1：不要因为短期波动而恐慌                              │
│   ─────────────────────────────                             │
│   • 中长线持仓波动是正常的                                   │
│   • 关注基本面，而非股价                                     │
│   • 短期下跌可能是加仓机会                                   │
│                                                             │
│   原则2：关注基本面变化                                      │
│   ─────────────────────────────                             │
│   • 定期检视公司基本面                                       │
│   • 基本面恶化才考虑卖出                                     │
│   • 基本面改善可以加仓                                       │
│                                                             │
│   原则3：定期检视，但不要频繁操作                            │
│   ─────────────────────────────                             │
│   • 每月检视一次即可                                         │
│   • 避免过度交易                                            │
│   • 不要被短期波动影响                                       │
│                                                             │
│   原则4：分批建仓，控制成本                                  │
│   ─────────────────────────────                             │
│   • 不要一次性买入                                          │
│   • 分3批左右建仓                                            │
│   • 利用下跌降低成本                                         │
│                                                             │
│   原则5：保持耐心，等待价值回归                              │
│   ─────────────────────────────                             │
│   • 价值投资需要时间                                         │
│   • 不要急于求成                                            │
│   • 相信时间的力量                                          │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

#### 20.4.2 中长线投资的常见错误

```
中长线投资常见错误
│
├── 短期思维
│   ├── 因为短期下跌就卖出
│   ├── 因为短期上涨就追高
│   └── 正确做法：关注基本面
│
├── 频繁操作
│   ├── 每天看盘，频繁交易
│   ├── 手续费吃掉利润
│   └── 正确做法：每月检视
│
├── 不分批建仓
│   ├── 一次性全仓买入
│   ├── 买在高点被套
│   └── 正确做法：分3批建仓
│
├── 不设止损
│   ├── 基本面恶化不卖出
│   ├── 越跌越补
│   └── 正确做法：逻辑止损
│
└── 缺乏耐心
    ├── 等不及价值回归
    ├── 频繁换股
    └── 正确做法：保持耐心
```
