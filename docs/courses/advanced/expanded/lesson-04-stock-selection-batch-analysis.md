## 4.1 为什么需要批量分析？

单股分析虽然详细，但每次只能分析一只股票。当你需要：

- 从一个行业中筛选最好的股票
- 定期检视自选股的状态
- 追踪某个热点板块的机会

这时候，批量分析就派上用场了。

**批量分析的优势**：

| 优势 | 说明 |
|------|------|
| **效率高** | 一次分析多只股票，节省时间 |
| **便于对比** | 同时看到多只股票的评分，便于筛选 |
| **发现机会** | 可能发现之前没注意到的股票 |
| **风险预警** | 及时发现自选股的风险变化 |

**系统限制**：
- 批量分析最多支持10只股票（系统限制）
- 超过10只需要分批提交

## 4.2 系统批量分析功能

### 4.2.1 访问批量分析功能

**操作步骤**：
1. 登录系统
2. 点击左侧菜单：`股票分析` → `批量分析`
3. 或直接访问：`/analysis/batch`

**页面布局**：
- 顶部：风险提示
- 左侧：股票列表输入区域
- 右侧：分析配置区域
- 底部：提交按钮

### 4.2.2 批量分析配置

**必填项**：
- **批次标题**：给这次批量分析起个名字（如："银行板块分析"）
- **股票代码列表**：输入要分析的股票代码（每行一个，最多10只）

**可选配置**：
- **批次描述**：描述本次分析的目的（可选）
- **分析深度**：1-5级（与单股分析相同）
- **分析引擎**：旧版引擎（稳定）或 v2.0引擎（推荐）
- **分析师团队**：选择需要的分析师（与单股分析相同）

**股票代码格式**：
- 支持多种格式：`000001`、`000001.SZ`、`600036.SH`、`AAPL`、`TSLA`
- 系统会自动识别市场类型
- 可以每行输入一个，也可以逗号分隔

## 4.3 批量分析的三大使用场景

### 4.3.1 场景一：行业扫描

**目的**：从某个行业中选出最值得投资的股票

**操作流程**：

**步骤1：确定目标行业**
```
例如：你看好银行板块
```

**步骤2：获取行业股票列表**
```
手动输入行业内主要股票代码：
000001（平安银行）
600036（招商银行）
601166（兴业银行）
600000（浦发银行）
601328（交通银行）
```

**步骤3：设置分析参数**
```
批次标题：银行板块分析
批次描述：筛选银行板块优质股票
分析深度：3级 - 标准分析（推荐）
分析引擎：v2.0引擎
分析师：市场分析师、基本面分析师、板块分析师
```

**步骤4：提交批量分析**
```
点击"提交批量分析"按钮
系统会创建批量分析任务
自动跳转到任务中心查看进度
```

**步骤5：查看分析结果**
```
在任务中心查看批量分析进度
等待所有任务完成（通常10-30分钟）
点击批次查看详细结果
按评分排序，找出评分最高的股票
```

**步骤6：深入研究**
```
对评分前3-5的股票进行单股深度分析
特别关注评分高且风险低的股票
```

**案例演示**：

假设你想从医药行业中选股：

```
批量分析结果示例（医药行业Top5）：

排名  股票代码  股票名称    评分  投资建议  风险等级
1    600276   恒瑞医药    85   买入      低
2    000538   云南白药    82   买入      低
3    300760   迈瑞医疗    80   持有      中
4    603259   药明康德    78   持有      中
5    002821   凯莱英     75   观望      中

你的行动：
→ 对评分前3的股票进行单股深度分析
→ 特别关注恒瑞医药（评分最高，风险低）
```

### 4.3.2 场景二：自选股定期检视

**目的**：定期检查自选股的状态变化，及时发现机会或风险

**操作流程**：

**步骤1：准备自选股列表**
```
从你的自选股中选择要检视的股票
建议每周检视一次观察池（10-20只）
```

**步骤2：创建批量分析任务**
```
批次标题：自选股周检视 - 2024-01-15
批次描述：定期检视观察池股票状态
股票代码：输入要检视的股票代码（最多10只）
分析深度：2级 - 基础分析（快速）
分析引擎：v2.0引擎
```

**步骤3：提交并等待完成**
```
提交批量分析任务
在任务中心查看进度
等待所有任务完成
```

**步骤4：对比分析结果**
```
查看各股票的评分、投资建议、风险等级
对比上次分析结果：
- 哪些股票评分上升了？→ 可能有机会
- 哪些股票评分下降了？→ 需要关注
- 哪些股票风险等级变化了？→ 重点关注
```

**步骤5：调整自选股**
```
评分持续下降的 → 考虑移出观察池
新发现的好股票 → 考虑加入观察池
状态稳定的 → 继续持有/观察
```

**自选股检视记录表**：

```
日期：2024-01-15

股票代码  上周评分  本周评分  变化  投资建议  备注
600519    88       90       +2   买入      基本面改善
000858    75       72       -3   持有      行业承压
002475    80       65       -15  观望      业绩预警！需重点关注
300750    70       78       +8   买入      估值修复，可考虑介入
```

### 4.3.3 场景三：热点追踪

**目的**：快速判断某个热点板块是否值得参与

**操作流程**：

**步骤1：发现热点**
```
从新闻、社媒、行情中发现热点板块
例如：AI概念股大涨
```

**步骤2：列出热点股票**
```
找出该板块的主要成分股（最多10只）
例如：AI概念股
- 000977（浪潮信息）
- 002230（科大讯飞）
- 300496（中科创达）
- 002415（海康威视）
```

**步骤3：快速批量分析**
```
批次标题：AI概念股快速分析
分析深度：1级 - 快速分析（追求效率）
分析师：市场分析师、新闻分析师、社媒分析师
```

**步骤4：判断热点质量**
```
查看分析结果：
- 大部分股票评分高 → 热点有基本面支撑
- 只有少数股票评分高 → 热点可能只是炒作
- 风险等级普遍高 → 追涨风险大
```

**步骤5：决策**
```
热点质量好 → 可以参与，选龙头，设好止损
热点质量差 → 观望或回避
```

**热点判断框架**：

| 热点类型 | 特征 | 参与策略 |
|---------|------|---------|
| **高质量热点** | 有基本面支撑，龙头清晰 | 积极参与，选龙头，设好止损 |
| **中等质量热点** | 有一定逻辑，但估值偏高 | 轻仓参与，快进快出 |
| **低质量热点** | 纯炒作，没有业绩支撑 | 回避或短线博弈（高风险） |

## 4.5 自选股管理方法

### 4.5.1 分层管理体系

一个有效的自选股管理应该采用分层管理：

```
┌─────────────────────────────────────────────────────────────┐
│                     自选股分层管理体系                        │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   ┌─────────────────────────────────────────────────────┐  │
│   │                 🌟 核心池（3-5只）                    │  │
│   │  ─────────────────────────────────────────────────  │  │
│   │  • 深度研究过的股票，非常了解                         │  │
│   │  • 重点跟踪，每日关注                                │  │
│   │  • 等待合适时机建仓，或已经持有                       │  │
│   │  • 入池标准：分析评分>80，你认可其投资逻辑            │  │
│   └─────────────────────────────────────────────────────┘  │
│                           ↑                                 │
│                         晋升                                │
│                           │                                 │
│   ┌─────────────────────────────────────────────────────┐  │
│   │                 📋 观察池（10-20只）                  │  │
│   │  ─────────────────────────────────────────────────  │  │
│   │  • 初步筛选出的股票                                  │  │
│   │  • 定期检视（每周批量分析）                          │  │
│   │  • 等待进入核心池或淘汰                              │  │
│   │  • 入池标准：分析评分>70，有一定投资潜力             │  │
│   └─────────────────────────────────────────────────────┘  │
│                           ↑                                 │
│                         晋升                                │
│                           │                                 │
│   ┌─────────────────────────────────────────────────────┐  │
│   │                 📦 备选池（20-50只）                  │  │
│   │  ─────────────────────────────────────────────────  │  │
│   │  • 广泛扫描的结果                                    │  │
│   │  • 批量分析发现的潜力股                              │  │
│   │  • 等待进一步筛选（每月整理）                        │  │
│   │  • 入池标准：有一定关注价值                          │  │
│   └─────────────────────────────────────────────────────┘  │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 4.5.2 自选股的生命周期

```
股票在自选股中的生命周期
│
├── 发现阶段
│   ├── 来源：行业扫描、热点追踪、他人推荐、自己发现
│   └── 动作：加入备选池
│
├── 观察阶段
│   ├── 条件：备选池中评分持续较高的股票
│   └── 动作：晋升到观察池，开始定期跟踪
│
├── 深研阶段
│   ├── 条件：观察池中表现突出，值得深入研究
│   └── 动作：进行单股深度分析，晋升到核心池
│
├── 建仓阶段
│   ├── 条件：核心池中出现合适的买入时机
│   └── 动作：根据系统提示择时买入
│
├── 持有阶段
│   ├── 动作：使用持仓分析功能跟踪
│   └── 参考：第五部分持仓管理篇
│
├── 卖出阶段
│   ├── 条件：触发止盈止损或其他卖出条件
│   └── 动作：卖出后进行复盘
│
└── 移出/保留
    ├── 表现好：保留在核心池继续跟踪
    └── 表现差：降级或移出自选股
```

### 4.5.3 定期检视流程

| 频率 | 操作 | 目的 | 预计时间 |
|------|------|------|---------|
| **每日** | 查看核心池股票的表现 | 跟踪持仓和准持仓 | 5-10分钟 |
| **每周** | 对观察池进行批量分析 | 发现机会和风险 | 15-30分钟 |
| **每月** | 整理备选池，调整各层 | 保持自选股质量 | 30-60分钟 |
| **每季度** | 全面复盘自选股表现 | 优化选股策略 | 1-2小时 |

## 4.7 本课小结与思考

### 本课要点回顾

1. **批量分析的优势**：效率高、便于对比、发现机会、风险预警
2. **三大使用场景**：行业扫描、自选股检视、热点追踪
3. **系统功能**：支持最多10只股票批量分析，在任务中心查看结果
4. **自选股分层管理**：核心池（3-5只）→ 观察池（10-20只）→ 备选池（20-50只）
5. **定期检视频率**：每日（核心池）→ 每周（观察池）→ 每月（备选池）

### 思考题

1. 选择一个你看好的行业，使用批量分析功能扫描，记录你的发现。
2. 整理你目前的自选股，按照分层管理体系重新分组。
3. 建立你的自选股检视流程，每周进行一次批量分析。
4. 思考：如何利用批量分析结果优化你的选股策略？
