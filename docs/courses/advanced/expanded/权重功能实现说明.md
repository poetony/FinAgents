## 📋 功能概述

权重功能已成功实现！系统会根据用户选择的分析师自动计算权重，并在提示词中明确标注，帮助AI重点关注高权重报告。

## 🔧 实现方式

### 1. 权重计算

**位置**：`core/agents/utils/weight_calculator.py`

**功能**：
- `calculate_report_weights()`: 根据用户选择的分析师和交易风格自动计算权重
- `format_weighted_reports_prompt()`: 格式化带权重的报告提示词

**权重计算逻辑**：
- 根据用户选择的分析师报告，自动判断交易风格（短线/中长线）
- 根据交易风格，应用预设的权重配置
- 如果核心报告不全，使用平均权重

---

### 2. 权重在提示词中的体现

**实现位置**：
- `core/agents/adapters/bull_researcher_v2.py`
- `core/agents/adapters/bear_researcher_v2.py`
- `core/agents/adapters/research_manager_v2.py`
- `core/agents/adapters/position/action_advisor_v2.py`

**提示词格式示例**：
```
【重要报告】（权重高，请重点关注）
- 市场分析（技术分析）（权重40%）：
[报告内容]

【次要报告】（权重低，仅供参考）
- 基本面分析（权重5%，仅供参考）：
[报告内容]

**权重说明**：
- 请重点关注高权重报告，这些报告对投资决策更重要
- 低权重报告仅作为补充参考，不应过度依赖
- 在综合判断时，请根据权重给予相应的关注度
```

## 📝 用户使用指南

### 如何影响权重？

1. **选择分析师**
   - 在单股分析时，选择需要的分析师
   - 系统会根据选择的分析师自动计算权重
   - 选择的分析师 → 系统自动计算权重 → 在提示词中标注权重百分比

2. **不选择分析师**
   - 不选择的分析师 → 不参与分析 → 权重为0%

3. **分析深度**
   - 深度分析（3-4级）→ 报告更详细 → 在综合判断时影响更大
   - 快速分析（1-2级）→ 报告较简略 → 在综合判断时影响较小

---

### 推荐配置

**短线交易**：
- ✅ 选择：技术、新闻、社媒分析师
- ❌ 不选：基本面分析师（权重会自动降低）
- 分析深度：1-2级（快速分析）

**中长线投资**：
- ✅ 选择：基本面、板块、风险辩论分析师
- ❌ 不选：新闻、社媒分析师（权重会自动降低）
- 分析深度：3-4级（深度分析）

1. **自动化**：无需手动设置权重，系统自动计算

4. **灵活性**：通过选择/不选择分析师来影响权重分配
