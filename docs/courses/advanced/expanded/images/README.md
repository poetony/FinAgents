## 📐 截图规范

### 尺寸要求
- **推荐宽度**：1200px - 1600px
- **最小宽度**：800px
- **格式**：PNG（界面截图）/ GIF（操作演示）

### 内容要求
- 隐藏敏感信息（账户、密码等）
- 使用示例数据，不使用真实交易数据
- 确保界面元素清晰可见
- 必要时添加标注（箭头、框选）

### 标注规范
- 使用红色（#FF0000）或蓝色（#0066CC）进行标注
- 箭头粗细：3px
- 框选线条：2px 虚线
