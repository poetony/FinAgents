## 🎯 扩写重点

### 核心理念贯穿

所有章节都要体现以下核心理念：

1. **可进化循环机制**：计划→选股→择时→执行→持仓管理→风险管理→复盘→改进→下一轮
2. **AI多智能体协作**：6大专业分析师并行工作，多空辩论，风险辩论
3. **短板识别方法**：通过AI复盘评分（timing_score, position_score, emotion_score, attribution_score）和报告（weaknesses, suggestions）识别短板
4. **持续改进路径**：基于复盘结果优化交易计划和系统配置
5. **工作流引擎应用**：如何创建和优化个人专属分析流程

### 每章必含内容

- ✅ 系统功能演示：每个概念都要有对应的系统操作演示
- ✅ AI赋能特色：突出AI多智能体协作的优势
- ✅ 可进化理念：强调如何通过复盘改进
- ✅ 实战案例：具体的操作步骤和案例
- ✅ 短板识别：如何通过复盘找出问题

1. **课程信息**：课时、目标

5. **实战案例**：具体的操作步骤
6. **小结与思考**：总结要点，布置思考题
