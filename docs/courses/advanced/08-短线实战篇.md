### 17.1 短线交易的特点

#### 17.1.1 什么是短线交易？

**短线交易定义**：持股周期 1天 - 1周

```
短线交易的时间框架
│
├── 超短线（T+1）
│   ├── 持股1-2天
│   ├── 追求快速获利
│   └── 风险和收益都较高
│
├── 短线（3-5天）
│   ├── 持股3-5天
│   ├── 捕捉短期波段
│   └── 最常见的短线模式
│
└── 波段短线（1周左右）
    ├── 持股5-10天
    ├── 捕捉较大波段
    └── 介于短线和中线之间
```

#### 17.1.2 短线交易的特点

```
短线交易的特点
│
├── 快进快出
│   ├── 资金周转快
│   ├── 不长期占用资金
│   └── 可以抓住更多机会
│
├── 技术为主
│   ├── 依赖技术分析
│   ├── 关注市场热点
│   └── 基本面权重较低
│
├── 时机敏感
│   ├── 对买卖时机要求高
│   ├── 差几分钟可能差很多
│   └── 需要快速决策
│
├── 时间投入
│   ├── 需要较多时间盯盘
│   ├── 关注盘中变化
│   └── 不适合没时间的人
│
└── 心理压力
    ├── 波动大，压力大
    ├── 需要良好的心态
    └── 容易情绪化交易
```

#### 17.1.3 短线交易适合什么人？

| 适合 | 不适合 |
|------|--------|
| 有时间盯盘 | 没时间看盘 |
| 反应快，决策果断 | 犹豫不决 |
| 心态稳定 | 容易情绪化 |
| 能承受较大波动 | 风险承受能力低 |
| 有一定技术分析基础 | 完全不懂技术分析 |

```
    ├── 关注突破信号

```

#### 17.3.2 短线分析报告解读

```
短线分析报告解读要点
│
├── 重点关注
│   ├── 技术形态（是否突破、是否回调到位）
│   ├── 市场热度（是否是当前热点）
│   ├── 资金流向（是否有资金流入）
│   └── 多空辩论（多方是否占优）
│
├── 次要关注
│   ├── 基本面（短线权重低）
│   └── 长期趋势（短线不太关注）
│
└── 决策依据
    ├── 技术面确认 + 热点配合 = 买入
    ├── 技术面破位 + 热点退潮 = 卖出
    └── 不确定 = 观望
```

### 17.4 短线交易的注意事项

#### 17.4.1 短线交易的五大纪律

```
┌─────────────────────────────────────────────────────────────┐
│                   短线交易五大纪律                            │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   纪律1：严格执行止损，不要死扛                              │
│   ─────────────────────────────                             │
│   • 短线止损要果断，触发就执行                               │
│   • 不要幻想反弹，不要死扛                                   │
│   • 小亏是正常的，大亏是致命的                               │
│                                                             │
│   纪律2：不追高，等回调再进场                                │
│   ─────────────────────────────                             │
│   • 追高容易被套在高位                                       │
│   • 等待回调到支撑位再买入                                   │
│   • 宁可错过，不可做错                                       │
│                                                             │
│   纪律3：热点退潮要果断离场                                  │
│   ─────────────────────────────                             │
│   • 热点持续性有限                                          │
│   • 退潮时不要恋战                                          │
│   • 落袋为安，寻找下一个热点                                 │
│                                                             │
│   纪律4：控制交易频率                                        │
│   ─────────────────────────────                             │
│   • 避免过度交易                                            │
│   • 没有机会就空仓等待                                       │
│   • 质量比数量重要                                          │
│                                                             │
│   纪律5：保持良好心态                                        │
│   ─────────────────────────────                             │
│   • 不急不躁，严格按系统执行                                 │
│   • 不被情绪左右                                            │
│   • 亏损后不要急于翻本                                       │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

#### 17.4.2 短线交易的常见错误

```
短线交易常见错误
│
├── 追高
│   ├── 看到涨停就追
│   ├── 结果买在高位
│   └── 正确做法：等回调
│
├── 死扛
│   ├── 亏损后不止损
│   ├── 幻想反弹
│   └── 正确做法：严格止损
│
├── 频繁交易
│   ├── 每天都要交易
│   ├── 手续费吃掉利润
│   └── 正确做法：等待机会
│
├── 重仓单只
│   ├── 把所有资金押一只
│   ├── 风险过于集中
│   └── 正确做法：分散仓位
│
└── 情绪化交易
    ├── 亏损后急于翻本
    ├── 盈利后过于自信
    └── 正确做法：保持冷静
```

### 18.3 择时阶段

```
大盘分析
│

│   ├── 中期趋势：上涨
│   └── 结论：大盘环境良好
```

#### 18.3.2 步骤2：确定买入时机

```
买入时机判断
│
├── 技术面确认
│   ├── 股价已突破前高
│   ├── 回踩5日均线获得支撑
│   └── 技术面确认突破有效
```
