### 1.1 为什么散户总是亏钱？

在A股市场，有一个广为流传的说法：**"七亏二平一赚"**——70%的散户亏钱，20%的散户不赚不亏，只有10%的散户能够盈利。

为什么会这样？让我们深入分析散户亏钱的三大根本原因：

#### 1.1.1 追涨杀跌的心理陷阱

**典型场景1：追涨**

> 小王看到某只股票连续涨停，朋友圈都在讨论，新闻也在报道。他心想："这股票这么火，肯定还会涨！"于是在第三个涨停板的时候追进去了。结果，第二天股票开始回调，小王被套在了高位。

**典型场景2：杀跌**

> 小李买入一只股票后，股价开始下跌。跌了5%的时候他想"再等等"，跌了10%的时候他开始焦虑，跌了15%的时候他终于受不了了，割肉卖出。结果，卖出后第二天股票就开始反弹了。

**这种行为的心理根源是什么？**

```
追涨杀跌的心理机制
│
├── 追涨心理
│   ├── 从众心理：别人都在买，我也要买
│   ├── FOMO（错失恐惧）：怕错过赚钱机会
│   └── 贪婪：想快速赚钱
│
└── 杀跌心理
    ├── 恐惧：怕亏更多
    ├── 止损无标准：不知道该在哪里止损
    └── 情绪失控：亏损带来的痛苦让人无法理性思考
```

**关键洞察**：追涨杀跌的本质是**没有预先设定的买卖规则**，完全被市场情绪和自己的情绪所左右。

#### 1.1.2 没有系统化的决策流程

让我们对比一下专业投资者和普通散户的决策过程：

**普通散户的决策过程**：

```
听说某股票不错 → 看了看K线图 → 感觉会涨 → 买入
     ↓
股价波动 → 不知道该怎么办 → 凭感觉操作
     ↓
亏钱或偶尔赚钱 → 归因于运气 → 没有总结
```

**专业投资者的决策过程**：

```
研究市场环境 → 筛选符合条件的股票 → 深度分析 → 制定交易计划
     ↓
按计划执行买入 → 设定止盈止损 → 按规则管理持仓
     ↓
按计划执行卖出 → 详细复盘 → 优化交易计划
```

**差距在哪里？**

| 维度 | 普通散户 | 专业投资者 |
|------|---------|-----------|
| **选股依据** | 听消息、看热点、凭感觉 | 有明确的选股标准和筛选流程 |
| **买入决策** | 看着顺眼就买 | 有明确的买入条件和时机 |
| **卖出决策** | 亏了就扛，赚了就跑 | 有预设的止盈止损规则 |
| **仓位管理** | 全仓进出 | 有科学的仓位分配策略 |
| **复盘总结** | 很少复盘 | 每笔交易都详细复盘 |

#### 1.1.3 情绪化交易的危害

**情绪对交易的影响是巨大的**。研究表明，在情绪激动的状态下，人的决策能力会下降50%以上。

**贪婪的表现**：
- 赚了10%还想赚更多，结果利润回吐
- 满仓一只股票，想"赌一把大的"
- 看到别人赚钱就急于跟进

**恐惧的表现**：
- 股价稍有波动就想卖出
- 不敢在低位买入（怕继续跌）
- 账户亏损时不敢看账户

**一个真实的案例**：

> 2020年3月，疫情导致A股大跌。很多散户在恐慌中割肉离场。但之后市场快速反弹，创业板指数在半年内涨了近50%。那些在恐慌中卖出的人，不仅亏了钱，还错过了反弹的机会。

**情绪化交易的根本问题**：没有预先制定好的规则来约束自己的行为。当情绪来临时，没有规则可遵循，只能被情绪控制。

### 1.2 什么是交易计划？

#### 1.2.1 交易计划的定义

**交易计划 = 一套完整的、可重复执行的投资决策规则**

更具体地说，交易计划是一套明确回答以下问题的规则集合：

```
┌─────────────────────────────────────────────────────────┐
│                      交易计划                            │
├─────────────────────────────────────────────────────────┤
│                                                         │
│   1. 买什么？（选股规则）                                │
│      └── 什么样的股票符合我的投资标准？                  │
│          • 行业选择标准                                  │
│          • 基本面要求                                    │
│          • 技术形态要求                                  │
│                                                         │
│   2. 什么时候买？（择时规则）                            │
│      └── 什么条件下触发买入？                           │
│          • 大盘环境要求                                  │
│          • 个股买点条件                                  │
│          • 确认信号要求                                  │
│                                                         │
│   3. 买多少？（仓位规则）                                │
│      └── 投入多少资金？分几批买入？                      │
│          • 总仓位控制                                    │
│          • 单只股票仓位上限                              │
│          • 分批建仓策略                                  │
│                                                         │
│   4. 怎么管理？（持仓规则）                              │
│      └── 持有期间如何应对各种情况？                      │
│          • 加仓条件                                      │
│          • 减仓条件                                      │
│          • 换股条件                                      │
│                                                         │
│   5. 什么时候卖？（止盈止损规则）                        │
│      └── 什么条件下触发卖出？                           │
│          • 止损条件和比例                                │
│          • 止盈条件和方法                                │
│          • 时间止损规则                                  │
│                                                         │
│   6. 如何改进？（复盘规则）                              │
│      └── 如何总结经验、优化系统？                        │
│          • 复盘频率                                      │
│          • 复盘内容                                      │
│          • 改进方法                                      │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

#### 1.2.2 为什么需要交易计划？

**1. 消除情绪干扰**

有了交易计划，你的决策就不再是"我觉得"，而是"规则显示"。

```
没有系统：股价跌了5% → 好慌啊，要不要卖？→ 纠结半天 → 做出情绪化决定
有了系统：股价跌了5% → 检查是否触发止损规则 → 没触发 → 继续持有
```

**2. 提供决策框架**

交易计划就像一个决策树，帮你在各种情况下做出一致的决策。

**3. 实现可重复性**

好的交易计划可以重复使用，让你的成功可以复制，而不是靠运气。

**4. 便于复盘改进**

有了明确的规则，你才能分析哪些规则有效、哪些需要改进。

#### 1.2.3 交易计划的特征

一个好的交易计划应该具备以下特征：

| 特征 | 说明 | 反例 |
|------|------|------|
| **明确性** | 规则清晰，没有模糊地带 | "感觉差不多就买" |
| **完整性** | 覆盖交易的各个环节 | 只有选股规则，没有止损规则 |
| **可执行性** | 规则可以具体执行 | "在合适的时候买入" |
| **一致性** | 每次都按同样的规则执行 | 这次止损5%，下次止损10% |
| **可优化性** | 可以根据复盘结果改进 | 规则过于模糊无法评估 |

#### 1.2.4 一个简单的交易计划示例

让我们看一个最简单的交易计划，帮助你理解什么是"规则"：

```
【简单趋势跟踪系统】

选股规则：
• 股价在20日均线之上
• 20日均线向上倾斜
• 最近5日成交量大于20日均量

买入规则：
• 股价回调到20日均线附近（±3%）时买入
• 大盘指数在20日均线之上

仓位规则：
• 单只股票仓位不超过30%
• 同时持有不超过3只股票
• 分2批买入，每批50%

止损规则：
• 股价跌破20日均线且收盘确认，止损卖出
• 最大亏损不超过8%

止盈规则：
• 盈利20%，卖出一半锁定利润
• 剩余持仓跟踪20日均线，跌破则清仓

复盘规则：
• 每周末复盘本周交易
• 记录每笔交易的理由和结果
```

**注意**：这只是一个示例，不是推荐的交易计划。重点是让你理解什么是"明确的规则"。

## 第2课：交易闭环概述

**课时**：1课时
**目标**：理解完整的交易闭环流程，了解各环节对应的系统功能

---

### 2.1 什么是交易闭环？

#### 2.1.1 闭环的概念

**交易闭环**是指从选股到复盘的完整交易流程，形成一个循环：

```
     ┌──────────────────────────────────────────────────────┐
     │                                                      │
     │                     交易闭环                          │
     │                                                      │
     │      ┌─────┐    ┌─────┐    ┌─────┐    ┌─────┐       │
     │      │选股 │───▶│择时 │───▶│执行 │───▶│持仓 │       │
     │      └─────┘    └─────┘    └─────┘    └──┬──┘       │
     │         ▲                                 │          │
     │         │                                 ▼          │
     │      ┌─────┐                          ┌─────┐       │
     │      │复盘 │◀────────────────────────│风控 │       │
     │      └─────┘                          └─────┘       │
     │         │                                            │
     │         │         反馈优化                           │
     │         └────────────────────────────────────▶      │
     │                                                      │
     └──────────────────────────────────────────────────────┘
```

**为什么叫"闭环"？**

因为复盘的结果会反馈到选股环节，形成一个闭合的循环。通过这个循环，你的交易计划会不断优化。

#### 2.1.2 闭环的六个环节详解

让我们详细了解每个环节：

**环节1：选股 - 买什么？**

这是交易的起点。你需要从几千只股票中，选出符合你标准的股票。

```
选股环节的核心问题：
├── 我要投资什么类型的股票？（成长股/价值股/题材股）
├── 我的选股标准是什么？（基本面/技术面/热点）
├── 如何筛选出符合条件的股票？
└── 如何对筛选出的股票进行深度分析？
```

**环节2：择时 - 什么时候买？**

选好股票后，不是立刻买入，而是要等待合适的时机。

```
择时环节的核心问题：
├── 当前大盘环境如何？是否适合入场？
├── 行业处于什么阶段？是否有机会？
├── 个股的买点在哪里？什么信号触发买入？
└── 多空双方的力量对比如何？
```

**环节3：执行 - 买多少？怎么买？**

确定买入后，需要决定投入多少资金，如何分批建仓。

```
执行环节的核心问题：
├── 这只股票我投入多少资金？（仓位）
├── 一次性买入还是分批买入？
├── 每批买入的价格和数量是多少？
└── 如何在模拟/实盘中下单？
```

**环节4：持仓管理 - 持有期间怎么办？**

买入后，股票会波动，你需要决定如何应对。

```
持仓管理的核心问题：
├── 股价上涨，是否加仓？加多少？
├── 股价下跌，是否减仓？减多少？
├── 出现重大消息，如何应对？
└── 什么情况下换股？
```

**环节5：风险管理 - 如何控制风险？**

这是贯穿全程的环节，但在持仓期间尤为重要。

```
风险管理的核心问题：
├── 我能承受多大的亏损？（止损位）
├── 我的盈利目标是多少？（止盈位）
├── 有哪些风险需要关注？
└── 如何设置止盈止损单？
```

**环节6：复盘 - 做对了什么？做错了什么？**

卖出后，交易并没有结束，复盘是非常重要的环节。

```
复盘的核心问题：
├── 这次交易，我的选股对吗？
├── 我的择时对吗？
├── 我的仓位管理对吗？
├── 我的止盈止损执行得如何？
├── 有什么经验可以总结？
└── 有什么教训需要吸取？
```

#### 2.1.3 闭环的意义

**1. 确保完整性**

很多散户只关注选股和买入，忽视了卖出和复盘。闭环思维确保你不会遗漏任何环节。

**2. 实现持续改进**

复盘的结果会反馈到选股和择时环节，让你的系统越来越好。

```
第1轮闭环：发现选股时过于看重短期热点，导致追高
    ↓
优化选股规则：增加估值筛选条件
    ↓
第2轮闭环：发现止损执行不到位，亏损扩大
    ↓
优化止损规则：设置自动止损单
    ↓
第3轮闭环：......
```

**3. 建立一致性**

闭环让你的每次交易都遵循相同的流程，避免随意性。

### 2.3 短线 vs 中长线的闭环差异

虽然闭环的结构相同，但短线交易和中长线投资在每个环节的侧重点不同：

#### 2.3.1 详细对比

| 环节 | 短线交易（1天-1周） | 中长线投资（1月-1年+） |
|------|-------------------|---------------------|
| **选股重点** | 技术形态、热点题材、资金流向 | 基本面、行业前景、估值水平 |
| **分析配置** | 技术面40%+新闻30%+情绪30% | 基本面50%+行业30%+风险20% |
| **择时依据** | 技术突破、热点启动、量价配合 | 估值低位、安全边际、政策利好 |
| **大盘要求** | 不能是明显下跌趋势 | 不能是熊市中期 |
| **仓位特点** | 集中（1-3只），单只可达50% | 分散（5-10只），单只不超20% |
| **建仓方式** | 分2批，快速建仓 | 分3批，缓慢建仓 |
| **止损比例** | 严格（3-5%） | 宽松（10-15%） |
| **止盈方式** | 固定目标或移动止盈 | 分批止盈 |
| **持仓管理** | 紧密跟踪，灵活调整 | 定期检视，关注基本面 |
| **复盘频率** | 每日复盘 | 每周/每月复盘 |
| **心理压力** | 较大，需要快速决策 | 较小，时间充裕 |

#### 2.3.2 如何选择适合自己的风格？

**适合短线的人**：
- ✅ 有较多时间盯盘
- ✅ 反应快，决策果断
- ✅ 能承受较大心理压力
- ✅ 对技术分析有一定了解
- ✅ 享受交易的刺激感

**适合中长线的人**：
- ✅ 时间有限，无法频繁盯盘
- ✅ 性格稳健，不急于求成
- ✅ 更关注公司价值而非股价波动
- ✅ 希望交易不影响正常生活
- ✅ 对基本面分析有兴趣

**重要提醒**：

> 没有绝对好坏之分，关键是找到适合自己的风格。本课程会分别讲解短线和中长线两种风格，你可以根据自己的情况选择，或者两种都尝试，最终找到最适合自己的。
