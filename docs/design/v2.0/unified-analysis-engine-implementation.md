## 🎯 项目概述

统一分析引擎是 TradingAgentsCN v2.0 的核心里程碑，成功将三套分散的股票分析流程统一为一套代码，实现了工作流编辑器与正式分析的完整闭环。

## 🏗️ 架构成果

### 核心组件

1. **UnifiedAnalysisService** - 统一分析服务
   - 位置: `app/services/unified_analysis_service.py`
   - 功能: 统一所有分析入口的调用逻辑
   - 支持: 工作流ID参数、进度回调、配置覆盖

2. **WorkflowEngine** - 工作流执行引擎
   - 位置: `core/workflow/engine.py`
   - 功能: LangGraph 动态构建和执行
   - 特性: 流式输出、状态管理、错误处理

3. **DefaultWorkflowProvider** - 默认工作流提供者
   - 位置: `core/workflow/default_workflows.py`
   - 功能: 系统默认工作流管理
   - 支持: 自动初始化、版本管理

4. **ReportFormatter** - 报告格式化器
   - 位置: `app/utils/report_formatter.py`
   - 功能: 统一输出格式、提取结构化数据
   - 集成: SignalProcessor、Markdown清理

## 🔧 技术细节

### API 统一
```
之前: 3套独立的分析流程
├── /api/workflows/{id}/execute (WorkflowEngine)
├── /api/analysis/single (TradingAgentsGraph)  
└── /api/analysis/batch (SimpleAnalysisService)

现在: 1套统一的分析引擎
└── UnifiedAnalysisService
    └── WorkflowEngine (LangGraph)
        └── 所有智能体统一调用
```
