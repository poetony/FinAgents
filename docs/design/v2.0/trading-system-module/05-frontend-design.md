# 前端交互设计

> 本文档定义个人交易计划模块的前端界面和交互流程

---

## 1. 页面结构

### 1.1 入口位置

```
侧边栏菜单
├── 📊 分析中心
├── 📈 模拟交易
├── 💼 持仓管理
├── 📋 操作分析
├── ⚙️ 我的交易计划  ← 新增入口
└── 👤 个人中心
```

### 1.2 页面层级

```
/trading-system                    # 交易计划列表页
/trading-system/create             # 创建向导
/trading-system/:id                # 系统详情/编辑页
/trading-system/:id/versions       # 版本历史页
```

### 6.3 API 调用

```typescript
// web/src/api/tradingSystem.ts
export const tradingSystemApi = {
  list: () => request.get('/api/trading-system'),
  getActive: () => request.get('/api/trading-system/active'),
  create: (data) => request.post('/api/trading-system', data),
  update: (id, data) => request.put(`/api/trading-system/${id}`, data),
  activate: (id) => request.post(`/api/trading-system/${id}/activate`),
  export: (id) => request.get(`/api/trading-system/export/${id}`)
}
