## 🔍 详细对比

### 工具1: 新闻工具

| 特性 | 旧版本 (`unified_news_tool.py`) | 新版本 (`stock_news.py`) |
|------|--------------------------------|-------------------------|
| **文件大小** | 596行 | 140行 |
| **架构** | 类方法 (`UnifiedNewsAnalyzer`) | 装饰器函数 (`@tool` + `@register_tool`) |
| **数据库缓存** | ✅ 支持MongoDB缓存 | ❓ 需要确认 |
| **数据源** | AKShare + Google新闻 | AKShare + Google新闻 |
| **市场支持** | A股、港股、美股 | A股、港股、美股 |
| **同步机制** | ✅ 复杂的线程池+事件循环处理 | ❓ 需要确认 |
| **注册方式** | 手动注册到toolkit | 自动注册 (`@register_tool`) |

**结论**: 
- ✅ 新版本更简洁（减少76%代码）
- ⚠️ 需要确认新版本是否保留了数据库缓存和同步机制
- ⚠️ 旧版本有更复杂的错误处理和日志记录

### 工具2: 技术指标工具

| 特性 | 旧版本 (`indicators.py`) | 新版本 (`technical_indicators.py`) |
|------|-------------------------|----------------------------------|
| **文件大小** | 354行 | 210行 |
| **架构** | 纯函数库 | 装饰器函数 (`@tool` + `@register_tool`) |
| **指标计算** | ✅ 独立的计算函数 | ✅ 使用stockstats库 |
| **支持的指标** | MA, EMA, MACD, RSI, BOLL, ATR, KDJ | MACD, RSI, KDJ, BOLL等 |
| **RSI风格** | ✅ 支持3种风格（ema/sma/china） | ❓ 需要确认 |
| **数据源** | 需要外部提供DataFrame | ✅ 自动获取数据（Tushare/AKShare/yfinance） |
| **注册方式** | 不注册（纯计算库） | 自动注册 (`@register_tool`) |

**结论**:
- ✅ 新版本集成了数据获取和指标计算
- ✅ 新版本使用成熟的stockstats库
- ⚠️ 旧版本的独立计算函数可能在其他地方被使用
- ⚠️ 需要确认新版本是否支持中国式RSI计算
